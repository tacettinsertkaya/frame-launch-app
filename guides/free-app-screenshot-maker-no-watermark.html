How to avoid generic screenshot assets

Start with the message, not the frame. Write one outcome-focused line for each screenshot, then choose the background and device style. Keep visual effects consistent across the set so the listing feels deliberate instead of assembled from unrelated images.

Export QA

Before publishing, export a single PNG and inspect it at small size. Check caption readability, frame alignment, contrast, and whether the first screenshot explains the app without needing surrounding copy. Then export the full ZIP.
